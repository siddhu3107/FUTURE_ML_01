# 📈 Sales & Demand Forecasting Project

## 🎯 Objective
This project predicts future sales using historical data and Machine Learning techniques. It helps businesses understand demand patterns and make better decisions.

---

## 📊 Dataset
Shampoo Sales Dataset (Monthly Time Series Data)

---

## ⚙️ Workflow
- Data Loading
- Data Preprocessing
- Train-Test Split
- Linear Regression Model
- Future Sales Forecasting
- Visualization

---

## 📈 Results
The model successfully predicts future sales trends and shows expected growth in demand for the next 6 months.

---

## 💡 Business Impact
- Inventory Planning
- Revenue Forecasting
- Reducing Overstock/Stockouts
- Better Business Decision Making

---

## 🛠 Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook
